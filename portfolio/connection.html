In deze week ging de les over databases, de soorten databases en de manier waarop deze databases gebruikt kunnen worden. Databases kunnen opgedeeld worden in 2 grote groepen:

  1. SQL of relational databases
    Deze databases zijn de populairste databases, dit heeft voornamelijk te maken met het feit dat deze makkelijker te gebruiken zijn en een stuk beter te begrijpen zijn. Een relationele database is een database die bestaat uit tabellen, elk met een schema waarin de data gepresenteerd wordt. De tabellen hebben allen een key die wordt gebruikt om specifieke tabellen, kolommen of rijen te identificeren en hier sneller toegang tot te krijgen. SQL of Structured Query Language databases hebben als voordeel dat deze zeer goed gestructureerd zijn. Dit zorgt ervoor dat deze databases eenvoudig te gebruiken zijn, hierdoor zijn er veel opgeleide mensen die met deze databases kunnen werken. Het nadeel van deze databases is dat ze niet werken met slecht gestructureerde data, dit maakt het lastig om dit databasetype te gebruiken voor bijvoorbeeld analyses van nieuwe data.
  2. NoSQL of Non-relational databases
    Dit type database is het populairste alternatief voor de SQL databases. Bij dit type database kan de database uit allerlei vormen bestaan in tegenstelling tot de SQL database. Dit heeft als groot voordeel dat de NoSQL database zeer wijd ingezet kan worden en flexibeler is in de toepassingen en het gebruik. Ook zijn NoSQL databases toleranter met foutjes in de databases. Echter hebben ook NoSQL databases een aantal nadelen, zo zijn deze databases door de vrijheid in gebruik ook lastiger toe te passen. Hierom worden deze databases nog minder de gebruikt dan SQL databases, echter is het gebruik van NoSQL steeds meer op komst wat duidt op een breder toepassingsveld.
